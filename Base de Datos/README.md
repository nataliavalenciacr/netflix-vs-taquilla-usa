# Base de Datos
Este repositorio contiene la base de datos utilizada para el análisis econométrico del efecto del crecimiento de suscriptores de Netflix sobre los ingresos semanales de taquilla en Estados Unidos (2015–2022).
## Archivo

base_streaming_taquilla USA.xlsx — Base de datos principal con las variables utilizadas en el modelo de regresión.

## Variables incluidas

Ingresos semanales de taquilla en EE.UU. (USD)
Número de suscriptores de Netflix
Variables de control (las que hayas incluido, como año, trimestre, etc.)

## Fuentes

Box Office Mojo / The Numbers — datos de taquilla
Statista / reportes de Netflix — datos de suscriptores

## Notas

Los datos fueron recopilados y organizados manualmente para el análisis.
El procesamiento y limpieza se realizó en R y Excel.
