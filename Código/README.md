# Código
Este archivo contiene el código utilizado para el análisis econométrico del efecto del crecimiento de suscriptores de Netflix sobre los ingresos semanales de taquilla en Estados Unidos (2015–2022).
## Archivo

código streaming vs taquilla — Script principal del análisis en R.

## Contenido del script

Carga y limpieza de la base de datos
Estadísticas descriptivas
Construcción del modelo de regresión lineal
Generación de tablas y gráficos de resultados
